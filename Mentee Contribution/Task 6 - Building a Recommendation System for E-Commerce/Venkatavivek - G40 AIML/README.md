# 🛍️ E-Commerce Recommendation System

## 📌 Project Overview
This project builds a machine learning-based recommendation system for an e-commerce platform. It combines regression, classification, and clustering techniques to analyze customer behavior and provide personalized recommendations.

---

## 🎯 Objectives
- Predict product ratings using regression
- Predict purchase likelihood using classification
- Segment customers using clustering
- Improve personalized recommendations

---

## 🧠 Techniques Used

### 🔹 Regression
- Model: Linear Regression / Random Forest
- Goal: Predict product ratings
- Metrics: MAE, RMSE, R²

### 🔹 Classification
- Model: Logistic Regression
- Goal: Predict purchase likelihood
- Metrics: Accuracy, Precision, Recall, F1-score, ROC-AUC

### 🔹 Clustering
- Model: K-Means
- Goal: Segment customers
- Metrics: Silhouette Score, Inertia

---

## 📊 Key Results
- Regression achieved good prediction accuracy (R² ~ 0.7+)
- Classification achieved strong performance (Accuracy ~ 80–85%)
- Optimal clusters identified: **3 customer segments**
- High-value customers identified based on behavior patterns

---

## 👥 Customer Segments
- **Cluster 0** → Low engagement customers  
- **Cluster 1** → Moderate customers  
- **Cluster 2** → High-value customers 💰
