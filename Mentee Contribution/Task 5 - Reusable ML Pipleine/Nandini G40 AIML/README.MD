# Customer Churn Prediction Pipeline

## Project Title
**Reusable Customer Churn Prediction Pipeline using Scikit-learn**

## Project Description

This project is developed to predict whether a customer is likely to churn from a subscription-based business. A reusable machine learning pipeline is created using Scikit-learn to automate data preprocessing, model training, evaluation, and prediction. The pipeline can also be saved and reused to make predictions on new customer data without repeating the preprocessing steps.

## Objectives

- Build a reusable machine learning pipeline.
- Preprocess customer data automatically.
- Train a Random Forest Classifier.
- Evaluate the model using different performance metrics.
- Save the trained pipeline for future use.
- Predict churn for new customers.

## Technologies Used

- Python
- Pandas
- Scikit-learn
- Joblib

6. Train the Model
7. Evaluate the Model
8. Save the Pipeline
9. Predict New Customer Churn

## Output

- Trained customer churn prediction pipeline
- Model evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix
- Saved `.pkl` pipeline file
- Prediction for new customer data

## Conclusion

This project demonstrates how a reusable Scikit-learn pipeline can simplify the machine learning workflow. It combines preprocessing and model training into a single pipeline, making the code cleaner, more organized, and easy to reuse for future customer churn predictions.
