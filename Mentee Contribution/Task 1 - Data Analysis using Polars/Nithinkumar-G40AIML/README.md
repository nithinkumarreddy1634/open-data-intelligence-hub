# Urbanization Analytics Engine

## Project Overview

The Urbanization Analytics Engine is a data analytics project developed using Python and Streamlit. It analyzes urban population growth and population density trends across different cities.

## Objectives

* Analyze urban population growth.
* Calculate population density.
* Visualize urbanization trends.
* Compare multiple cities using interactive dashboards.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Streamlit

## Project Structure

Urbanization_Analytics_Engine/

├── data/

│ └── urbanization_data.csv

├── app.py

├── analysis.py

├── requirements.txt

└── README.md

## Features

* Interactive city selection
* Population growth visualization
* Population density calculation
* Data table display
* Dashboard interface

## How to Run

1. Install required libraries:
   pip install pandas numpy matplotlib streamlit

2. Navigate to the project folder.

3. Run the application:
   python -m streamlit run app.py

## Expected Output

The dashboard displays:

* Population growth trends
* Population density analysis
* City-wise urbanization statistics
