# Multi-Algorithm Recommendation Comparison

## Project Overview
This project compares three machine learning algorithms using Scikit-learn.

## Algorithms Used
- Ridge Regression
- Logistic Regression
- K-Means Clustering

## Hyperparameter Tuning
- GridSearchCV

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score
- Accuracy
- Precision
- Recall
- F1 Score
- Inertia
- Silhouette Score

## Conclusion
The three algorithms were implemented and evaluated using different performance metrics. Hyperparameter tuning with GridSearchCV was used to improve model performance.
